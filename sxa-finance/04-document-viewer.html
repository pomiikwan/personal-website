1. 策略概述 / Strategy Overview

本策略基于动量突破原理，通过识别价格突破关键技术水平，结合成交量确认和多重时间框架分析，在趋势初期建立仓位。策略适用于A股市场，主要针对中小市值股票的短线交易机会。

策略目标年化收益率: 18%-25%，最大回撤控制在 12% 以内。

2. 入场规则 / Entry Rules

2.1 主信号 / Primary Signal

  • 价格突破20日最高价（HIGH(20)）
  • 当日成交量 > 20日平均成交量的 1.5倍
  • RSI(14) 在 50-75 区间内
  • MACD 柱状图为正值且递增

2.2 确认条件 / Confirmation

// 动量突破入场条件伪代码
func checkEntrySignal(bar: Bar) -> Signal {
    let breakout = bar.close > HIGH(20)
    let volumeConfirm = bar.volume > SMA(bar.volume, 20) * 1.5
    let rsiRange = RSI(14) > 50 && RSI(14) < 75
    let macdConfirm = MACD().histogram > 0

    if breakout && volumeConfirm && rsiRange && macdConfirm {
        return Signal.LONG_ENTRY
    }
    return Signal.NONE
}
