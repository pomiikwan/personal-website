快速开始

1. 获取API密钥

联系客服获取API访问密钥，每个密钥有调用次数限制

免费版：100次/天 专业版：1000次/天

2. 发起API调用

在请求头中携带API密钥，使用标准HTTP客户端调用

Authorization: Bearer YOUR_API_KEY
Content-Type: application/json

3. 处理响应数据

所有接口返回统一JSON格式，包含code、message、data字段

{
  "code": 200,
  "message": "success",
  "data": { ... }
}

4. 错误处理

遇到错误时检查code字段，参考文档获取详细说明

{
  "code": 401,
  "message": "Invalid API key",
  "data": null
}
